#  Flutter Dependency Injection & State Management Guide

> A beginner-friendly guide to understanding **GetIt**, **Injectable**, and **BLoC** pattern in Flutter

##  What's Inside

This repository contains comprehensive notes and examples on how to properly implement:

- **Dependency Injection (DI)** using GetIt and Injectable
- **State Management** using BLoC pattern
- **Clean Architecture** principles
- **API Client Setup** with Dio

## Who Is This For?

Junior Flutter developers who want to understand:
- Why we use dependency injection
- How GetIt and Injectable work together
- How to implement BLoC pattern correctly
- How all the pieces fit together in a real app
